uint64_t odp_system_meminfo_t::total_overhead

Total memory usage overheads.

Total amount of memory (in bytes) that is currently consumed by roundings to alignment/block/page size limits, etc. overheads. This is a subset of 'total_used' bytes. Zero if not supported by the implementation.
